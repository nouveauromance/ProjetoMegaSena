# Projeto Mega Sena

- Aplicativo mobile desenvolvido como atividade da disciplina de Programação para Dispositivos Móveis.

## Sobre o projeto

- O Projeto Mega Sena é um aplicativo mobile que simula o funcionamento de um sorteio de loteria inspirado na Mega-Sena.
O sistema realiza o sorteio de 6 números aleatórios, no intervalo de 1 a 60, garantindo que:

  - Não haja repetição de números;

  - Os números sejam exibidos na tela em ordem numérica;

  - O usuário possa realizar novos sorteios.

## Escopo

- O aplicativo possui as seguintes funcionalidades:

  - Botão Sortear:
    - Gera 6 números aleatórios entre 1 e 60;
    - Impede números duplicados;
    - Atualiza o resultado na tela.

  - Botão Limpar:
    - Remove os números exibidos;
    - Reinicia o sorteio.

## Instalação

- Instale o projeto com npm:

```bash
  npm install ProjetoMegaSena
  cd ProjetoMegaSena
  npm start
```
